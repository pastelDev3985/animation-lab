# 🎬 Animation Lab

A structured, hands-on journey into mastering modern web animations using real code, experiments, and mini-projects.

---

## Overview

This repository documents my learning process as I explore and implement animations across different levels—from foundational CSS animations to advanced interactive UI experiences.

Rather than passively learning, this repo focuses on **building, testing, and refining animation techniques** that can be applied in real-world applications.

---

## What This Repository Covers

### 1. CSS Animation Fundamentals

* Transitions
* Transforms
* Keyframes
* Hover effects

### 2. React Animations

* Component-based animations
* Reusable motion patterns
* UI interactions

### 3. Framer Motion

* Page transitions
* Scroll-triggered animations
* Staggered animations
* Gesture-based interactions

### 4. Animated Components

* Navigation bars
* Cards & hover effects
* Modals & overlays

### 5. Mini Projects

* Animated landing pages
* Interactive UI sections
